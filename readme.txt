Кэширующий ДНС Сервер.
Для запуска сначала устанавливаем библиотеку с помощью pip install dnslib, используется для парсинга днс ответа и формирования днс запроса.
Затем запустить сервер можно:
 python dnsserver.py
Затем сервер может принимать запросы
Для тестирования имеется файл testdns, после запуска сервера запускаем тестовый файл
  python testdns.py
Он отправляет нашему днс серверу запрос для имени vk.com.
Сервер при получении ответа заносит запись в кеш, и при повторном запросе уже выдаст ответ из кеша.
После завершения работы днс сервер сериализует весь кеш в файл cash.txt. Файл создается сам.
